## About this build
- Arabic-first, RTL single-page static site (no backend).
- Luxury gold/black art direction with a Greek-key (meander) frieze signature drawn from the shop's own storefront sign.
- Mobile-first, tasteful 60fps motion (IntersectionObserver reveals, sticky condensing header), `prefers-reduced-motion` respected.
- WhatsApp CTA → the business's own number. Google Maps embed + directions.
- Fonts: Amiri (Arabic display), Tajawal (Arabic body), Cormorant Garamond (Latin accents).

## Assets
Local photos only, in `/assets/` (never hotlinked). Source had 5 files but photo-2/3/4 were byte-identical duplicates, so 3 distinct images are used:
